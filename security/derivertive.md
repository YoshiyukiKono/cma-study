# デリバティブと投資戦略

## 先物理論価格

配当を無視する場合（月）
```
F = S * {1 + r * N/12}
```
* S: 原資産価格
* r: 短期金利（年率）
* N: 満期までの月数

配当を考慮する場合（年）
```
F = S * {1 + (r - d)T}
```
* d: 配当利回り（年率）
* T: 満期までの期間（年）
