# コーポレート・ファイナンスと企業分析

## 加重平均資本コスト Weighted Average Cost of Capital（CAPM成立時）
```
r_WACC = E/(E+D) * r_E + D/(E+D) * (1 - T) * r_D
```

## 資本コストの推定: MM理論（デフォルトリスクがない場合）
```
β_L = β_U [1+(1-T) * D/E]
```

## 自己資本コスト
```
r_E = r_F + β_E(r_M - r_F)
```

## 資本コスト
```
資本コスト = FCF / 企業価値
```
上記の形で、資本コストを問われることがある。
上記式の解釈は直感的でないが、要は以下の式の変形。
```
企業価値 = FCF / 資本コスト
```
