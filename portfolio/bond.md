## キャリー・ロールダウ

### 意義
* キャリー: （短期金利を差し引いた）利息収入
* ロールダウン: （順イールドの下での）利回り低下による価格上昇

両者のトータルを考慮して比較することで、どの利付債に投資するかを検討する。

### 計算方法
* キャリー: クーポン分 - 金利コスト
* ロールダウン: 利回り低下による価格上昇分をデュレーションで近似計算
```
(y(T) - r) * ⊿t
```
* 残存期間T年
* パーイールド: y(T)
* 修正デュレーション: D(T)
* 保有期間（年）: ⊿t
```
D(T) * (y(T) - y(T - ⊿t))
```
