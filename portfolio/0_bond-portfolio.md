# 債券ポートフォリオ

|収益の源泉|仮説・アノマリー|
|:----|:----|
|長期債キャリー|流動性プレミアム仮説|
|ロールダウン|特定期間選考仮説|

## DF

```
DF_n = 1 / (1 + r_0,n)^n
```
DFがあれば、計算時にスポットレートの代わりになる。

```
(1 + r_0,n)^n = 1/DF_n
```
## キャリー・ロールダウン

```
キャリー = (y(T)-r)*Δt*(1/D(T))
```
* y(T): 残存期間Tのパー・イールド
* r: 短期金利
* Δt: 債権の保有期間
* D(T): 修正デュレーション

つまりは、残存期間T年のパー・イールド（例えば国債利回り）から短期金利を引き、修正デュレーションで割る（1年以上のキャリーをもとめられているなら、年数を掛ける）

```
ロールダウン = y(T) - y(T - Δt)
```
残存期間10年と残存期間5年の利回りが示されている場合は、それらの差分を取り、その差の期間（5年）で割る。

```
超過リターン = キャリー + ロールダウン * 1年後想定修正デュレーション
```
