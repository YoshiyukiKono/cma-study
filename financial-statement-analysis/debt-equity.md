## ICレシオ Interest Coverage Ratio

http://fsreading.net/analysis/018.shtml

どの程度余裕を持って営業利益で借入金の利息をまかなえているかを示す指標

```
ICレシオ = 事業利益（営業利益 + 金融収入）÷ 支払利息
```
